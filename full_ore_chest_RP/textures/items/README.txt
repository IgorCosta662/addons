# TEXTURE PLACEHOLDER FILES
# 
# Para adicionar texturas customizadas, crie arquivos PNG 16x16 com os seguintes nomes:
# 
# - chest_iron_full.png       (Baú com detalhes cinza/ferro)
# - chest_gold_full.png       (Baú com detalhes dourados)
# - chest_diamond_full.png    (Baú com detalhes azul-claro/ciano)
# - chest_coal_full.png       (Baú com detalhes pretos/cinza escuro)
# - chest_netherite_full.png  (Baú com detalhes roxo/netherite)
#
# Coloque os arquivos PNG nesta pasta: textures/items/
#
# Dica: Você pode editar a textura do baú vanilla e adicionar uma cor/brilho
# para indicar visualmente qual tipo de minério o baú contém.
#
# Texturas recomendadas:
# - Use o baú normal como base
# - Adicione uma camada de cor correspondente ao minério
# - Adicione brilho/partículas para indicar que está "cheio"
